・素質強化について
　ほぼオマケ。
　最強四重絶頂というロマンを求めて淫○シリーズの素質を復活させました。重複取得可。
　○○狂いのように能力上限があがったりしませんが、○敏感相当の感度補正がかかります。
　絶頂強度が１段階くらい上にあがります（元々高めの絶頂を出せるＶは補正が低めになっています）
　また、取得していると対応能力の絶頂時に体力と気力の減少が半減しますが、気休めレベルです。

・Ｖ、Ａはともかく、残念ながらＢとＣはこれに敏感と器具つけても最強絶頂には到達しません。
　しかし１つの素質を強力にしすぎるのも問題なんで、このままにしておきます。
　もう一段階何かあれば到達できる気がする…
