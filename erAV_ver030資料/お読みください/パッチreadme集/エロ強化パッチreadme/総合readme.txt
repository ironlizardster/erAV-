eraIm@sP0.02b用エロ強化パッチ＋まとめ

・このパッチは参考にeratohoYMをプレイしてみたらあまりの機能の多さに驚いた自分が
　eram@sに移植しようと試みたものです。
　エロを強化するのを拒否する理由など無い。
　自分のパッチと響パッチとIWSパッチ最新版、ABLUPの修正を統合した内容になっています。
　パッチ作者様は以降これをベースにしていただければ幸いです。
　流用元のパッチ作者様を含め、偉大なる先人達に多謝！

・使い方
　ERBとCSVを全部上書きして下さい。全部です。各パッチのすり合わせをしているので
　半端にいれると何かが動かなくなります。また、バックアップは必ず取って置いてください。
　Emuera起動推奨。

・内容物
　新規のパッチについて説明します。詳しくは別記のテキストを読むといいでしょう。
　新規のパッチは以下の４点

　・失神パッチ…対象が失神するようになります
　・追加コマンド…新たなるコマンドが使えるようになります
　・精飲絶頂パッチ…対象が精液を飲むと絶頂するようになります。
　　　　　　　　　　近日アップされていたerapoke用の精飲絶頂パッチを流用させていただきました
　・素質強化…感度の上がる素質が習得できるようになります
　
　以上は全てコンフィングでオンオフの切り替えができます。デフォルトではオンになっているので注意してください

・他
　研究室でふたなり、母乳体質、おもらし癖を直せるようになりました。
　金の力は偉大と言う事で。
　追加コマンド用の地の文を書いた時に、本当にちょろっとだけ他の部分もいじっています。
　フェラチオの後に舐め取る文が出ないのを直したりとか、セックスの大量射精用地の文を追加したり、
　絶頂の文を一段階増やしてみたり…
　パイズリの屈服刻印を２相当に下方修正しました。
