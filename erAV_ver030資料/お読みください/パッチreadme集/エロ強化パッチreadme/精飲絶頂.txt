・精飲絶頂パッチ
　対象が精液を飲むと絶頂するようになります。YMからの移植を考えていましたが、erapoke用の精液絶頂パッチを
　作った方がいたので、そちらを流用させていただきました。簡潔にまとめてあったので非常に助かりました！
　
・基本はerapoke版精飲絶頂の流用ですが、YMと同じように精愛味覚を習得できるようになっています。
　口内射精時に絶頂していると、強制精飲絶頂となり、一定回数以上の強制精飲絶頂をすると精愛味覚を習得し、
　以降刺激が無くても精液を飲むだけで精飲絶頂となります。

・精愛味覚取得の目安は強制精飲絶頂回数が大体５０回前後（キャラの素質によって前後します）
　取得時に精液中毒が３に上がり、精液中毒のレベルが若干上げやすくなります。
　また、精愛味覚を所持しているとフェラ系のコマンドを行った時に体力と気力の消費を低減できます（大体３分の２くらい）
