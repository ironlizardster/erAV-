・失神パッチについて
　eratohoYMの物を流用しています。流用といってもバリアントいじりなどした事無い私にはまともに動かすのも
　一苦労でした。一番手間のかかった部分です。マジで大変だった。
　
・調教対象に一定以上の快楽、苦痛、恐怖を与えるとランダムで失神します。以下元readmeより引用

失神中は
・コマンド選択時の実行判定が行われません（コマンドを拒否されない）。
・調教対象が条件を満たしていなくても一部のコマンドを選択出来るようになります。
・逆に一部コマンドが選択出来なくなります。
・口上が表示されません（テキストは一部を除き表示される）。
・一部パラメータが上がりません。
・刻印が付きません。
・失神から回復した時は通常のパラメータ上昇に加え、一部パラメータにボーナスが付きます（良い方にも悪いに方も）（この辺りは結構大雑把）。
・失神から回復した時のパラメータ変動により通常と同様の刻印付加判定が行われます。
・条件を満たすとランダムで失神から回復します。
・条件を満たさなくても調教を終了すると回復します。
