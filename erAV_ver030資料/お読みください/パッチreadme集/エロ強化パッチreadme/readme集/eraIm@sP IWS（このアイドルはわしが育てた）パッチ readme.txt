========================================================================================
eraIm@sP0.02b用 IWSパッチ（このアイドルはワシが育てたパッチ）　ver0.33
========================================================================================

※導入方法・仕様だけでも読んでやってください。

【導入方法】
・CSV・ERB両フォルダを本体フォルダに上書きしてください。
・（重　要）
　Ver0.2までを適用している人はSYSTEM_AWS.ERBを削除してください。
　お手間をおかけしてスミマセン…
・セーブデータに適用する場合は、コンフィグ→IWSシステム→CSVファイルの読み込みで
　スリーサイズ情報が読み込まれます。

【仕様】
・導入するとステータス画面にキャラのスリーサイズが表示されるようになります。
　（公式で発表されているキャラのみ）
・バストのカップサイズを表示します。
　計算には「美少女、バストカップ数測定スクリプト」を使用しています。
・AWS（このアイドルはわしが育てた）システムによりアイドルのおっぱいを育成することが出来ます。
　ぶっちゃけera☆staのAWS丸パクリです。
　コンフィグから設定することでB絶頂時におっぱいが成長します。
・なぜか涼ちんも成長します。愛の前には性別などさしたる問題ではありません。
　しかしながら絶壁持ちは成長しません。それをせいちょうさせるなんてとんでもない！
・妊娠時には徐々にスリーサイズが変化します。絶壁持ちでもバストが成長します。母は強し。
・母乳体質の人は人間牧場へ放牧することが出来ます。
　母乳を売ればポイントゲット！おっぱいもおっきくなるよ！やったねたえちゃん！

【さらに細かい仕様】
・小鳥さんが不明で悲しい、って人はコンフィグで設定してください。
　IWSシステムから手動で設定することが出来ます。
　非現実的な体型がお好みの方はCSVを直接弄ってください。
　ランダム生成を組み込もうかとも思いましたが絶対悲しいことになるので止めました。
　各種サイズはBASE:20〜36に保存されています。（下記の使用フラグ参照）
・AWS用のおっぱい素質を導入しています。
　ただしこれらを取得しても【爆乳】は残るようにしてありますし、
　（ステータス画面では表示されなくなります）
　取得してもなんらメリットはありません。気分です。
・研究室での魔改造は従来の「素質を変化させる」ものではなく、
　「カップサイズを１個引き上げる」仕様になっています。
　効果は薄くなり、値段も高いですが無制限に改造できます。
・妊娠によるスリーサイズ変化時にはAWS強度に応じてバスト成長値にボーナスが付きます。
　ボーナス分は出産育児後も維持される仕様です。
・IWSオフ時はスリーサイズ表示以外のすべての動作をスルーします（たぶん）。
　表示上のスリーサイズは変化しません（たぶん）。
・放牧できる条件は娼館とあまり変わりありませんがおっぱいがおっきくないとあんまり意味無いかも。
　ただし放牧するとおっぱいが成長するのでそのうち稼げるようになるかもしれません。
　えらく時間かかりますけども。
・IWSの強度が強すぎるとゲームバランスを崩すかも知れません。
　Lv1で使ってる限りは殆どネタの範疇で済みますので基本的にソレをお勧めします。
