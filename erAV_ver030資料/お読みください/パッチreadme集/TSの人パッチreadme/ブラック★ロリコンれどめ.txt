【仕様】
・ロリコン
	ロリコンフラグが一定数たまると、マスターに【ロリコン】がつきます。
	【ロリコン】がついた状態でロリコンフラグが一定数以下になると外れます。
	気持ちですが、調教効果がつきます。
	コンフィグでロリコンシステムをはずすことができます。（初期値はON）
	クリア後、【ロリコン】およびフラグをはずすこともできます。
・常時発情
	調教開始時に、発情と潤滑が3000からスタートします。
	あと、気持ちですが、調教効果もつきます。
	コンフィグではずすこともできます（初期値はON）
	ちなみに取得条件ですが、調教終了時における発情および潤滑をそれぞれ10000で除した数の累積がともに1500以上です。
	ただし、調教終了時に10000以上にならなかった場合は、累積がリセットされます。
	相当きつい割にメリットが薄そうなので、条件は変える可能性があります。
・性豪
	気持ちだけ調教効果がつきます。
	取得条件は、強化性癖をすべてとることです。
	まぁ、名誉職だと思ってください。

【修正したファイル】
・【ロリコン】【常時発情】【性豪】を追加（Talent.CSV）
・【ロリコン】【常時発情】【性豪】の取得設定（GET_SPECIAL_TALENT.ERB）
・【ロリコン】【常時発情】のON/OFFを制御（SYSTEM_CONFIG.ERB）
・強くてニューゲームのときにロリコンフラグの引継ぎをするかどうかを制御（SYSTEM_NEWGAME.ERB）
・【常時発情】の取得条件判定の累積値を取得・保存（TRAIN_MAIN.ERB）
・調教効果を気持ちだけ追加（SYSTEM_SOURCE_SUB1.ERB）

【直したバグ】
・一定の性癖を持つ者が露出癖Lv5以上のとき、痴情の珠がなくても露出癖をレベルアップできる不具合を解消（ABLUP17.ERB）
・反発刻印の減少で、ここだけ「宝珠」となっていたので、「珠」に修正（ABLUP99.ERB）
