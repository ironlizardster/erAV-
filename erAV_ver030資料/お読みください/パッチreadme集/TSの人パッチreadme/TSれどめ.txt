【仕様】
導入すると、研究所で200,000ポイント支払うことにより、性転換できます。
「オトコ」フラグがある場合は「オトコ」フラグと「童貞」フラグが消滅し、「処女」フラグが立ちます。
「オトコ」フラグがない場合は「オトコ」フラグと「童貞」フラグが立ち、「処女」フラグと「ふたなり」フラグが消滅します。
異常経験が1増えます。
性転換を3回繰り返すと淫乱が取れてしまうので、一度転換すると元に戻すことはできません。
記憶を消しても性別は転換したままです。
