eraIm@sP0.02b用カレーパッチver0.2

【注意点】
　カレーパッチver.0.1ではFLAG:80〜FLAG:82を月日の管理用に使っていましたが、ver0.2以降はDAY:日付_月〜DAY:日付_季節に変更しています。
　以前のセーブデータを移行した場合はコンフィグから月日を再設定してください。(この時FLAG:80〜FLAG:82も0にします)

【概要】
　このパッチはアイマスキャラの年齢・誕生日・血液型の設定をeraIm@sPに取り込むためのものです。
　誕生日の判定に伴って月日の表示・更新処理も行ないます。
　特定の日付でイベントを発生可能にしました(現状、バレンタインのみ)。
　曜日・季節の表示も行なっていますが、現状では「単に表示しているだけ」です。

　eraIm@sP0.02b用エロ強化パッチ＋まとめを導入した環境でご使用ください。

【使い方】
　eraIm@sP0.02b用エロ強化パッチ＋まとめ(era4519)を導入したうえで、ERBとCSVを全部上書きして下さい。
　また、作者が初心者なためデバッグ不足な可能性は高いです。
　バックアップは必ず取って置いてください。
　Emuera起動推奨です。

・2月14日の朝に好感度200以上のキャラが居る場合、バレンタインのイベントが発生します。←New!
・【小学生】の導入に伴って、進学時の処理を修正しました。←New!

・導入するとメイン画面で月日、ステータス画面で年齢・誕生日・血液型が表示されます。
・公式で実年齢が明かされていないキャラクター(ピヨちゃんとか)は、外見年齢と見た目の年代で
　表示されます。
　※erapokeのキャラソートをほぼそのまま流用しています。

・各キャラの誕生日(正確には、その前日から日付が変わった翌朝)に告知が表示されますが、それ
　による特殊イベント等は未実装です。

・コンフィグ画面から現在の月日、各キャラの年齢・誕生(ryを変更することができます。
　誕生日で年齢を増加させないことも可能です。
　※ぶっちゃけ、IWSシステムの処理をほぼそのまま流用しています。
　※ただし幾つかの処理をカットしているため、あなたや既に年齢データがあるキャラでも変更が
　　可能です。具体的に言えば小鳥さんじゅうろくさいができるよ！

・中学校/高校への進学処理は月日・年齢の導入に伴って変更しました。
　購入されていないキャラクターは年齢が変わらない為、亜美真美が律子と同学年、とかいう事も
　いずれ起き得ます。

・年齢順のキャラソートを可能にしました。実は一番時間を食った部分だったり。
　ソートが降順のみなので昇順もできるようにしたいのだが…今の自分には無理だorz

【使用フラグ】
　BASE:11　誕生月
　BASE:12　誕生日
　BASE:13　血液型(0=不明,1=A,2=B,3=O,4=AB)

　FLAG:74　誕生日・血液型システムの使用(OFFでも誕生日の処理で実年齢が増えないだけ)

　DAY:日付_月(旧FLAG:81)　現在の月
　DAY:日付_日(旧FLAG:82)　現在の日
　DAY:日付_年		　　現在の曜日(0=月,1=火,2=水,3=木,4=金,5=土,6=日)
　DAY:日付_季節(旧FLAG:80)　現在の季節(0=春,1=夏,2=秋,3=冬)
